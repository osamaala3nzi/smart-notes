# Smart Notes App

A simple and clean notes web application built with Flask and SQLite.

## Features
- Add notes
- Automatically categorize notes
- Delete notes
- Clean and simple user interface

## Tech Stack
- Python
- Flask
- SQLite
- HTML
- CSS

## How to Run
1. Install Flask:
   pip install flask

2. Run the app:
   py app.py

3. Open in browser:
   http://127.0.0.1:5000

## Project Purpose
This project was built to practice full-stack web development using Python and Flask, including backend routing, database integration, and frontend design.
